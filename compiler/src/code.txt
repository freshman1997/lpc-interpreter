

            1、运行时不会检查类型信息，编译时检查参数类型和数量是否能够对的上，运行时调用其他对象的函数不会检查参数类型、数量是否能够对的上，没传的参数补 0 （undefined）
            2、一个文件编译产生一个二进制文件，所以是可以多线程同时编译的，只要编译没有发生冲突和重复编译即可
            3、常量表、指令、行数信息、全局变量表存储在类上，函数原型仅保留开始指令位置、结束指令位置、参数数量，全局变量的赋值动作早于 create 函数调用，所以一个对象最开始执行的位置肯定是全局变量开始赋值的位置；
            4、函数存储在以上几种数据的后面，包含的数据有：（1）函数名称，（2）开始指令位置，（3）结束指令位置，（4）闭包信息
            5、闭包信息：（1）捕获的变量信息：a、局部变量数量，b、捕获变量为拷贝的形式，如果是指针那么会改变捕获的变量的内容，否则不会改变；（2）其他信息和正常函数一样
            6、类二进制文件结构如下：前缀统一为 [tag][length]
                    |       类名称                                          
                    |    类信息（函数数量，变量数量，常量表数量，行数信息数量）
                    |       class 信息，包含字段数量，class名称
                    |       函数信息
                    |       常量表
